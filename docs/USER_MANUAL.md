# 頁面內容即時編輯系統 (Live Edit) 操作手冊

## 1. 系統簡介
本系統專為管理員設計，可直接在網頁介面上即時修改文字與圖片，無需任何程式碼操作。具備自動存檔、版本控制與二階段發布驗證，確保內容異動的安全與穩定。

## 2. 角色與權限
*   **管理員 (Admin)**：具備編輯、發布、還原版本的所有權限。
*   **一般使用者 (Member/Guest)**：僅能看到已發布的正式內容，無法進入編輯模式。

## 3. 操作流程

### 3.1 進入編輯模式
1. 登入管理員帳號 (`admin/admin`)。
2. 導航至任何想要修改的頁面 (如首頁、個人簡歷等)。
3. 頁面右下角會出現一個深藍色懸浮面板，點擊「**開啟編輯模式**」。

### 3.2 編輯內容
*   **修改文字**：直接點擊頁面上的標題 (H1-H3)、段落 (P) 或簡介內容，即可像使用 Word 一樣直接輸入。
*   **更換圖片**：點擊頁面上的任何圖片，系統會跳出視窗要求輸入新的圖片網址。
*   **自動存檔**：系統每 5 秒會自動儲存一份草稿至您的瀏覽器中，避免意外關閉視窗導致內容遺失。

### 3.3 發布變更
1. 編輯完成後，點擊懸浮面板上的「**發布變更**」。
2. 系統會要求輸入管理員二階段驗證碼 (預設為 `8888`)。
3. 驗證通過後，草稿將正式生效，所有訪客皆可看到更新內容。

### 3.4 結束與捨棄
*   若不想保留目前的編輯內容，點擊「**結束並捨棄**」，系統將清除草稿並還原至目前的正式版本。

## 4. 版本管理 (VCS)
1. 進入 [CMS 控制台 (cms.html)](cms.html)。
2. 下拉至「**頁面內容版本管理**」區塊。
3. 選擇要管理的頁面 (如 `index.html`)。
4. 點擊歷史版本旁的「**還原**」按鈕，即可將該頁面恢復至過去的狀態。
